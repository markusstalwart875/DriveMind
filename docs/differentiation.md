# How DriveMind Differs

## Not just a diary tool
A diary records. DriveMind changes future execution.

## Not just a memory plugin
Memory stores context. DriveMind decides what should be remembered and reused.

## Not just a persistence prompt
Simple persistence can become reckless. DriveMind combines persistence with safety and judgment.

## Not just agent roleplay
Roleplay affects tone. DriveMind affects execution behavior over time.

## Core difference
DriveMind treats agent reliability as a product problem, not a prompt trick.
