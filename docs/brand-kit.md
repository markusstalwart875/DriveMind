# DriveMind Brand Kit

## Core positioning
DriveMind is a safety-and-reliability layer for AI agents.

## Core keywords
- safety
- reliability
- judgment
- persistence
- memory continuity
- reusable learning

## What to emphasize
- real work, not just chat
- reliability, not just intelligence
- bounded persistence, not blind pushing
- reusable experience, not just logs

## What to avoid
- sounding like a prompt gimmick
- sounding like roleplay or persona packaging
- sounding like a diary-only tool
- sounding reckless or overly aggressive

## Tone
- calm
- precise
- product-level
- practical
- builder-friendly

## Default tagline
DriveMind — A safety-and-reliability layer for AI agents

## Alternate taglines
- DriveMind — Helping agents move from usable to reliable
- DriveMind — Persistent execution, safer judgment, reusable memory
- DriveMind — Agents that remember, persist, and know when to stop
