# Example Session

## User
Enable DriveMind in execution mode for this task. Do not give up too early, but stop if the risk boundary becomes unclear.

## Expected behavior
1. Clarify the objective.
2. Keep pushing through the first failure with bounded retries.
3. Report blockers instead of silently stalling.
4. Escalate if the task would cross a risky boundary.
5. End with a short review and reusable lesson.

## Why this matters
This example shows that DriveMind is not only about persistence.
It is persistence plus safety plus reusable learning.
