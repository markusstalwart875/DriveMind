## Read this file only when
- the active question is whether the work should continue, switch path, or escalate
- authorization, irreversibility, or external impact is central

If not, read `boundary-preservation.md` instead.

---

## Core gate

Ask:
1. continue?
2. switch path?
3. escalate?

If the answer is unclear, do not improvise authority.
